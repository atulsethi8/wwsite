# WanderW - Your Travel Companion

A beautiful travel website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Modern gradient backgrounds
- Interactive elements
- Travel-focused content

## Deployment

This website is deployed on Netlify with continuous deployment from the main branch.

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Start developing!

## Technologies Used

- HTML5
- CSS3
- JavaScript (coming soon)
